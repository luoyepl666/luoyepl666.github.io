hexo 安装教程

发表于2024-04-14|更新于2023-09-06|网站搭建
|阅读量:

不知道发什么就把我安装博客的教程当成我的第一篇博客发布吧

安装博客流程

cmd进入你指定的目录后使用命令安装Hexo

输入hexo init初始化生成文件后安装主题与必备插件这里我用的是butterfly 具体指令看主题的github文档吧

输入命令 hexo generate可以缩写hexo g 生成HTML页面。 hexo cl && hexo g && hexo s 三板斧每次生成都要来一遍

部署项目到Github远程仓库 在hexo根目录中

创建密钥并上传密钥

发布到github

More info:

创建密钥那步的同时已经将生成的密钥添加到git仓库里了

下一步就是在hexo根目录 输入 hexo d 发布到仓库中

发布成功后等几分钟你就可以访问你的github用户名.github.io的个人博客了

不过因为github主机在国外，所以国内可能访问不了所以最好同步到gitee平台上
